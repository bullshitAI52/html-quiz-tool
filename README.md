# 智能题库系统使用说明

这个工具是一个离线版的网页刷题软件，无需安装即可使用。

## 1. 如何使用
直接双击文件夹中的 `index.html` 文件，它会默认使用你的浏览器（如 Chrome、Edge、Safari）打开。

## 2. 如何导入题库
系统默认是空的，你需要导入 Excel 格式的题库才能开始刷题。

1.  在网页右上角点击 **"导入新题库"** 按钮。
2.  选择电脑上的 Excel 文件 (`.xlsx` 或 `.xls`)。

### Excel 格式要求
您的 Excel 文件第一行必须是标题（表头），请确保包含以下列名：

| 列名 | 说明 | 示例 |
| :--- | :--- | :--- |
| **类型** | 必填。区分题目类型 | 单选、多选、判断 |
| **题干** | 必填。题目的内容 | 1+1等于几？ |
| **选项A** | 选择题必填 | 1 |
| **选项B** | 选择题必填 | 2 |
| **选项C** | 可选 | 3 |
| **选项D** | 可选 | 4 |
| **答案** | 必填。正确答案 | A 或 AB 或 对 |
| **解析** | 可选。题目解析 | 1+1=2 |

*注意：列名必须完全匹配（特别是"类型"、"题干"、"答案"）。*

## 3. 主要功能
*   **练习模式**：按顺序做题，做完一题显示一题的答案和解析。
*   **考试模式**：模拟考试，有 90 分钟倒计时。单选/多选/判断题混合，交卷后自动评分。
*   **错题库**：
    *   **练习错题库**：练习模式中做错的题会自动进入这里。
    *   **考试错题库**：考试中做错的题会自动进入这里。
*   **背题模式**：点击"切换到背题模式"，可以直接查看正确答案，方便快速记忆。
*   **数据保存**：您的答题进度和导入的题库会保存在浏览器缓存中，关闭网页下次打开还在。
