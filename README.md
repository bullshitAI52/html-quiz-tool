# 智能题库系统使用说明

这是一个**增强版**的离线网页刷题工具，支持图片、填空题、问答题等高级功能。

## 📦 版本说明

本项目提供两个版本：
- **`index.html`** (增强版) - 当前主版本，支持图片、填空题、问答题等高级功能
- **`index-basic.html`** (基础版) - 简化版，仅支持单选、多选、判断题

## 1. 如何使用

直接双击 `index.html` 文件，它会在浏览器中打开。

## 2. 如何导入题库

1. 点击页面中的 **"选择文件"** 或 **"导入"** 按钮
2. 选择您的题库文件（支持 `.xlsx`, `.xls`, `.csv` 格式）

### 增强版支持的格式

增强版可以导入包含以下列的 CSV/Excel 文件：
- **基础列**：问题、答案、选项A-F、题目类型、题目分数、答案解析
- **图片列**：问题图片、选项A-F图片
- **填空题列**：填空项1-6、填空项分值1-6
- **问答题列**：完整答案、关键词1-5、关键词分值1-5、评分规则

详见 `tools/题.csv.txt` 示例文件。

### 基础版格式要求

如果使用 `index-basic.html`，Excel 文件需包含以下列：

| 列名 | 说明 | 示例 |
| :--- | :--- | :--- |
| **题干** | 必填 | 1+1等于几？ |
| **答案** | 必填 | A 或 AB 或 对 |
| **选项A-H** | 选择题需要 | 1, 2, 3, 4 |
| **解析** | 可选 | 1+1=2 |
| **难度** | 可选 | 简单/中等/困难 |
| **分类** | 可选 | 数学 |

## 3. 主要功能

### 增强版特性 (index.html)
- ✅ **支持图片题目**：题干和选项都可以带图片
- ✅ **填空题**：支持多个填空项，每项可设置分值
- ✅ **问答题**：支持关键词评分
- ✅ **答题卡**：右侧显示答题进度，可快速跳转
- ✅ **即时反馈**：选完立即显示对错
- ✅ **详细统计**：答题分析、正确率统计

### 基础版特性 (index-basic.html)
- **练习模式**：按顺序做题，显示答案和解析
- **考试模式**：90分钟倒计时，自动评分
- **错题库**：练习/考试错题分类管理
- **背题模式**：直接查看答案，快速记忆
- **数据保存**：浏览器本地缓存

## 4. 题库制作工具

### 在线编辑器
访问 **`editor.html`** 或在线地址使用可视化题库制作工具：

**功能特点：**
- 📝 可视化表单，无需手动填写 Excel
- 🖼️ 支持上传图片（题干图片、选项图片）
- 🎯 支持多种题型（单选、多选、判断、填空、问答）
- 👀 实时预览题目效果
- 💾 导出为 CSV 格式，可直接导入答题系统

**使用流程：**
1. 打开 `editor.html`（或访问在线地址）
2. 填写题目信息（题干、选项、答案、解析）
3. 如需添加图片，点击"添加图片"按钮上传
4. 点击"添加到题库"保存当前题目
5. 重复步骤2-4添加更多题目
6. 完成后点击"导出 CSV"下载题库文件
7. 在答题系统（`index.html`）中导入下载的 CSV 文件

### 格式说明

参考 `tools/题.csv.txt` 示例文件，增强版支持的完整列格式：
- 问题、问题图片
- 选项A、选项A图片、选项B、选项B图片...（最多选项H）
- 答案、题目类型、题目分数、答案解析
- 填空题：填空项1-6、填空项分值1-6
- 问答题：完整答案、关键词1-5、关键词分值1-5、评分规则

## 5. 工具文件夹

`tools/` 目录包含辅助工具：
- **题库制作工具.html**：可视化题库编辑器（已部署为 `editor.html`）
